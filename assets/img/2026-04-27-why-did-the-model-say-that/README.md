# Media Directory for Blog Post

## How to Add Images

Place your images in this directory, then reference them in your blog post using:

```markdown
![Image Description](/blog/assets/img/2025-10-16-why-did-the-ai-say-that/your-image-name.jpg)
```

## Supported Formats
- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- SVG (.svg)

## Naming Conventions
Use descriptive names with hyphens:
- `gradcam-visualization.png`
- `tsne-embedding-space.png`
- `attention-weights-heatmap.png`
